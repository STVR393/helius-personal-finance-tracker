- File name: `docs/screenshots/planning-forecast.png`

- Goal: show why Helius is more useful than a basic ledger

## Capture Rules

- Use fake or scrubbed data only.
- Remove or mask your Windows username and full DB path.
- Keep the same terminal theme, scale, and window size across all shots.
- Prefer one clean full-screen window over cropped fragments.
- If you create a GIF later, show one short flow: add transaction -> budget update -> planning forecast change.
